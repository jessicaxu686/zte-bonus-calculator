# ZTE奖金最优分配计税计算器

## 项目简介
这是一个专为ZTE员工设计的奖金计税优化计算器，帮助员工合理规划年终奖和季度奖的提取方式，实现税后收益最大化。

## 功能特色
- 🧮 智能计算最优奖金分配方案
- 📊 对比不同计税方式的税后收益
- ⚠️ 临界点提醒，避免多交税
- 📱 响应式设计，支持移动设备
- 🎯 专为ZTE员工量身定制

## 使用说明
1. 输入全年总收入预估
2. 输入年终奖和季度奖金额
3. 查看最优分配建议
4. 对比不同方案的税后收益

## 技术特点
- 纯前端实现，无需服务器
- 使用Tailwind CSS框架
- 响应式设计
- 实时计算，即时反馈
