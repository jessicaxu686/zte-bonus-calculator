# 一次性奖金最优提取测算计算器

# 项目简介

这是个人创建的奖金计税优化计算器，帮助员工合理规划年终奖提取方式，实现税后收益最大化。

## 功能特色

* 🧮 智能计算最优奖金分配方案
* 📊 对比不同计税方式的税后收益
* ⚠️ 临界点提醒，避免多交税
* 📱 响应式设计，支持移动设备
* 🎯 纯前端本地计算工具，无数据传输和存储行为

## 使用说明

1. 按照说明输入各项数据
2. 查看最优分配建议
3. 对比不同方案的税后收益
4. 此工具根据需要提取的奖金总数进行一次性提取的建议，可以根据综合所得税率表临界值决定是否全部提取

## 免责声明

计算结果仅供参考，实际报税请以税务机关核定为准。
